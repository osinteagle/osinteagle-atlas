# OSINTEAGLE Geo

OSINTEAGLE Geo ist ein webbasiertes Geo-Analyse-Tool für Open Source Intelligence (OSINT).
Es ermöglicht das Erfassen, Analysieren und Verifizieren von geografischen Daten direkt im Browser.

Projekt: https://geo.osinteagle.com/

## Features

1. Daten importieren
- CSV (Lat/Lng)
- GeoJSON (Punkte, Linien, Polygone)
- Manuell per Karte
- Bearbeiten & Löschen via Sidebar oder Marker

2. Experten-Suche (Overpass API)
- OSM Query Builder
- Kombinierbare Filter
- Drag & Drop
- Aktiv/Deaktivierbar
- TagFinder API (Hover)
- Farbwahl

3. Karten & Layer
- Dark, Light, OSM, Satellit
- Heatmap
- Zeitslider
- Einstellungen werden gespeichert

4. Review & Fallbericht
- ✓ Geprüft / ★ Treffer
- Notizen
- HTML Export

5. Sichtkegel & Verifikation
- Richtung, Winkel, Reichweite
- Verifikations-Board
- HTML Export + Street View

6. Kontext-Panel
- Street View, Satellit
- Links zu Maps, Mapillary etc.
- Sonnenstand (Azimut, Elevation)

7. Koordinaten
- DD, DMS, DDM, UTM, MGRS
- Auto-Erkennung
- Copy Funktion

8. Export
- CSV, GeoJSON, KML, JSON
- Verifikations-Board als HTML

## Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP
